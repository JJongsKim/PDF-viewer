# PDF 뷰어 - 파싱

PDF를 업로드하고, 원하는 페이지의 콘텐츠를 파싱해봅시다!

## [프로젝트 실행 방법]

```
yarn install

⬇️

yarn start
```

## [Commit Convention]

| 머릿말   | 설명                         |
| -------- | ---------------------------- |
| Feat     | 기능 단위 구현               |
| Style    | 프로젝트 UI 퍼블리싱         |
| Fix      | 프로젝트 수정 건에 대한 처리 |
| Refactor | 코드 리팩토링                |
| Docs     | 문서 작업                    |
| Chore    | 기타 작업                    |

## 프로젝트 기능
- PDF 업로드 및 미리보기
- 업로드한 PDF 텍스트 파싱
